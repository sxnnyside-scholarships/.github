## 🧃 What this is ✦

Tools for teachers, systems students, and beginners  
who are just starting out ✦

Not market hypotheses. Not adoption statistics.  
Real problems — observed, understood, then solved (たぶん)

This branch doesn't build for the passive student.  
It builds for whoever teaches and for whoever learns with intention.  
So the software becomes invisible and what remains is the process.

## 🧠 The Stack ✦

<p align="center">
  <img src="https://skillicons.dev/icons?i=php,laravel,vue,dart,flutter,python,bun,svelte&theme=dark" />
</p>

<details>
<summary>✦ What's in the stack and why</summary>

<br>

**PHP / Laravel** — backend for teacher-facing platforms that need to just work  
**Vue** — reactive interfaces without unnecessary complexity  
**Dart / Flutter** — multiplatform reach for mobile learning tools  
**Python / PyQt** — desktop utilities with zero installation friction  
**Bun / Elysia.js** — fast, lightweight backend for real-time tools  
**Svelte** — frontend when performance and simplicity both matter

</details>

The stack follows the classroom.  
Not the other way around.
